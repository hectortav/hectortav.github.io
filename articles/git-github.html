Git and Github for beginners What is Git?
Git is a version control system for source code development. It is a big part of modern day programming and a necessary tools for every team and individual. Git was created by Linus Torvalds, who is also the creator of Linux.
Getting started
  • Windows
    • Download the Git Bash terminal:
    https://gitforwindows.org/

  • Ubuntu
    • Open a terminal and run the following commands:
    sudo apt-get update
    sudo apt-get install git
    git --version

  • Mac OS
    • Open a terminal and run the following commands:
    brew install git
    git --version

    Let's get started
    Set global git options:
    git config --global user.email "johndoe@example.com"
    git config --global user.name "John Doe"
    Create an empty repository:
    git init
    Stage files:
    # Add all files
    git add *
    # Add specific files
    git add file_name
    Unstage files:
    # Unstage all files
    git reset *
    # Unstage specific files
    git reset file_name
    You can make files invisible to git by creating a file called .gitignore. The "dot" at the beginning of the filename, means the .gitignore is a hidden file. To show hidden files in Ubuntu, you will need to use the -a flag ( ls -a ).To ignore a file, you just need to add it in a new line of the .gitignore file.

    A .gitignore example:

    # Ignore passwords.txt
    passwords.txt
    # Ignore all files with the ".exe" suffix
    *.exe
    Commit staged changes:
    git commit -m "Example commit message"
    See past commits:
    # List of all commits
    git log
    # Commits in a more readable way
    git log --oneline
    # Display only last 5 commits
    git log -n 5
    What is Github?
    Github is a website that hosts Git repositories in the cloud. Github helps developers manage their projects and collaborate on private and open-source projects.
    Create a new repository
    After you create a Github account, navigate to https://github.com/new where you can create a new repository. Choose a name and whether you want your project to be Public or Private. For your first project leave "Add a README file", "Add .gitignore" and "Choose a license" unchecked. Click "Create repository"!

    Now copy the repository link (mine is https://github.com/hectortav/test) and let's get back to the terminal:
    # Connect your local git repository with the remote github repo
    git remote add origin https://github.com/hectortav/test
    # Push your local changes to github
    git push origin master
