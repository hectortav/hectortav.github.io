ยง Introduction

 Elixir is a pretty new language, but it is based on the, not that new, Erlang Virtual Machine. Elixir is a functional language and it is used by big companies such us Discord, PepsiCo, and Facebook. I was fascinated by how easy it is to create scalable applications using Elixir and also by how fun it is to use. The first thing I tried to create using Elixir was a simple REST API, so that is what I will try demonstrate, how you can create a simple json API with Elixir and a minimal amount of effort.

ยง Setup

 For this project I am using linux, since Elixir and Postgres are easier to use on linux. Specifically, I am using Ubuntu 18.04.5 (WSL).

To install Elixir, open a terminal and run:
wget https://packages.erlang-solutions.com/erlang-solutions_1.0_all.deb
sudo dpkg -i erlang-solutions_1.0_all.deb
sudo apt-get update
sudo apt-get install elixir


To install Phoenix:
mix archive.install https://github.com/phoenixframework/archives/raw/master/phoenix_new.ez

For PostgreSQL:
# install
sudo apt install postgresql postgresql-contrib
# start
sudo service postgresql start
# access postgres
sudo su - postgres
psql
# create user 'postgres' with password 'postgres'
CREATE USER postgres WITH ENCRYPTED PASSWORD 'postgres'
# exit postgres
\q
exit

Lastly create the project:
mix phx.new project_name --no-webpack --no-html
